Getting started

For now, just as a reminder, here are some command's to work with:

To rebuild the docker containers (in case of new requirements for example):

docker-compose build

To run all docker containers:

docker-compose up

To run the web container but with dependencies pushed to the background:

docker-compose run --service-ports web

To run django migrations:

docker-compose run web python manage.py makemigrations docker-compose run web python manage.py migrate

Now for the cool stuff, your stuff. To create your own app you have to run:

docker-compose run web django-admin startapp spindlechannels.your_app_name

And add your new app to installed_apps in `spindlechannels/settings/base.py`.
